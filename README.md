# WhatEnvIs

A simple tool of os and browser detection.

## detect what

* os: ios/android/macos/windows
* device: phone/tablet/kindle/pc
* browser: ie/chrome/firefox/opera/safari
* app: wechat(ios/android)/alipay(ios/android)
